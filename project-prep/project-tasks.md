## Scenario Requirements
- Entirely **remote**.
- User Endpoints:
  - Windows
  - Linux
- Video editors will need a *large local backup solution*.
- Cloud-based data storage solution for video files.
- Written Collaboration 
  - Ability to work together over the cloud.

## Guideline Reqs.

1. User provisioning
2. Backups
3. OS Diversity
4. Collaboration
5. Remote Access
